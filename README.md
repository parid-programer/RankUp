# RankUp 🏆
> Prove Your Knowledge, Climb the Ranks.

RankUp is a highly-gamified, full-stack adaptive testing platform built to test your global knowledge across various subjects. Featuring a dynamic question engine powered by AI, real-time leaderboard aggregation, and an Elo-style XP ranking system, RankUp makes learning and trivia intensely competitive.

---

## 🚀 Features

- **AI-Powered Adaptive Testing**: Questions are algorithmically generated by OpenAI, dynamically scaling in difficulty (Levels 1-10) based on your performance and current Subject XP. We employ advanced system prompting to ensure well-rounded questions specifically avoiding highly repetitive concepts.
- **Custom Testing Engines**: Take strict preset 1m, 3m, or 5m tests, or use the precise Custom Duration engine (Minimum 30s) for a tailored session.
- **Gamified Ranking System**: Climb through 7 prestigious tiers: Bronze, Silver, Gold, Platinum, Diamond, Master, and Grandmaster.
- **Subject-Specific Tracking**: Play globally, or compete individually in Categories like Mathematics, Science, History, Geography, and Computer Science.
- **Stealth Penalties**: Rage-quitting or closing your browser abruptly guarantees a stealth database penalty. Don't run! The UI won't visually punish you, but the database will.
- **Global Leaderboards**: A heavily optimized, flat-structure MongoDB leaderboard showing the Top 50 sharpest minds alive. 🦗 (Look out for easter eggs!)
- **Robust Identity & Security**: Features Native NextAuth integration, passwordless Magic Links, 2-Factor Authentication (TOTP & Email), and customizable avatars powered by a local CDN pipeline.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 App Router + React 19
- **Database**: MongoDB & Mongoose
- **Authentication**: NextAuth.js
- **Styling**: Tailwind CSS + DaisyUI
- **Animations**: Framer Motion
- **AI Generation**: OpenAI API (`gpt-3.5-turbo`/`gpt-4`)
- **Security**: BcryptJS, Speakeasy (TOTP 2FA), Nodemailer

## 📦 Getting Started

### 1. Clone & Install
```bash
git clone https://github.com/yourusername/rankup.git
cd rankup
npm install
```

### 2. Environment Configuration
Create a `.env.local` file in the root directory and configure the following variables:

```env
# Database
MONGODB_URI="mongodb+srv://<user>:<password>@cluster.xyz.mongodb.net/rankup"

# Authentication
NEXTAUTH_SECRET="your_ultra_secure_random_string"
NEXTAUTH_URL="http://localhost:3000"
GOOGLE_CLIENT_ID="your_google_oauth_client_id"
GOOGLE_CLIENT_SECRET="your_google_oauth_client_secret"

# Test Engine Config
OPENAI_API_KEY="sk-your-openai-api-key"
NEXT_PUBLIC_TIME_DEDUCTION_RATE="1"

# Email Provider (SMTP for Password Reset & Email 2FA)
SMTP_HOST="smtp.mailprovider.com"
SMTP_PORT="587"
SMTP_SECURE="true"
SMTP_USER="noreply@yourdomain.com"
SMTP_PASS="your_smtp_password"
```

### 3. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
